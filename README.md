# Finanzas

programa que ayuda a gestionar las finanzas del usuario registrando todos los movimintos financieros que realice

## Descripcion
Es un porgrama pensado para ayudar al usuario a gestionar su dinero guardando lso movimientos economicos que realice el usuario guardandolos en una base de datos local y registrando la fecha

## funciones
* **SQLite** para la base de datos
* **OS** para crear la carpeta
* **datetime** para poder guardar la fecha
* **POO** codigo hecho utilizando la POO (programacion orientada a objetos)

## Tecnologias
* **Lenguaje:** Python
* **Sistema:** Arch Linux
* **Editor:** Neovim

## como ejecutarlo
para correr este programa, asegurate de tener instalado python y ejecuta:
''' bash
python3 finanzas.python'''
